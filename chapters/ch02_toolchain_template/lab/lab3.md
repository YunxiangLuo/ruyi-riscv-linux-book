# Lab 3: Makefile 工程模板、调试记录与性能边界

## 实验目标

- 创建课程统一工程模板。
- 编写 Makefile。
- 记录一次失败和修复过程。

## 实验环境

| 项目 | 要求 |
| --- | --- |
| 主机环境 | Linux、macOS 或 WSL |
| 目标环境 | QEMU 或开发板可选 |
| 软件依赖 | `make`、C 编译器、Git |

## 实验任务

### 任务 1：创建工程模板

建立 `src/`、`include/`、`build/`、`logs/`。

### 任务 2：编写 Makefile

实现 `build`、`run`、`clean`。

### 任务 3：记录调试过程

制造一个构建错误，保存失败输出，修复后再次构建。

## 实验步骤

```bash
make build
make run
make clean
git status
```

## 运行验证

| 验证项 | 预期现象 | 是否通过 |
| --- | --- | --- |
| 工程目录 | 结构清楚 |  |
| Makefile | 三个目标可执行 |  |
| 调试记录 | 有失败、判断、修复和成功输出 |  |

## 常见问题

| 现象 | 可能原因 | 处理方式 |
| --- | --- | --- |
| make 目标无效 | 缩进或目标名错误 | 检查 tab 和目标名 |
