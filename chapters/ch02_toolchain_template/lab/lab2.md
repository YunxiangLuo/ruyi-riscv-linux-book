# Lab 2: 工具链、Hello World 与 ELF 观察

## 实验目标

- 编译并运行 Hello World。
- 使用 `file`、`readelf` 和 `ldd` 观察程序。
- 记录一次运行失败或依赖差异的排查过程。

## 实验任务

1. 编写 Hello World。
2. 执行编译命令并保存输出。
3. 在目标系统运行程序。
4. 使用 `file`、`readelf -h` 和 `ldd` 查看 ELF 和依赖信息。
5. 如果运行失败，记录失败命令、返回值、错误输出和排查步骤。

## 运行验证

| 验证项 | 预期现象 | 是否通过 |
| --- | --- | --- |
| 程序运行 | 输出 Hello World |  |
| `file` | 能看到文件类型和目标架构 |  |
| `readelf` | 能看到目标架构 |  |
| `ldd` | 能看到依赖库或说明静态链接 |  |
| 调试记录 | 有失败、判断或差异说明 |  |

## 提交要求

- 源码。
- 编译命令。
- 运行、观察和失败排查结果。
