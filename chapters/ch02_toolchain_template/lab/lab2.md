# Lab 2: 统一工程模板与基准测试

## 实验目标

- 建立统一工程目录。
- 用 Makefile 管理构建。
- 记录一次基准测试结果。

## 实验任务

1. 创建课程工程模板。
2. 编写 Makefile。
3. 运行 CoreMark 或 Dhrystone。

## 运行验证

| 验证项 | 预期现象 | 是否通过 |
| --- | --- | --- |
| `make build` | 构建成功 |  |
| `make clean` | 清理成功 |  |
| benchmark | 输出测试结果 |  |

## 提交要求

- 工程目录截图或树形结构。
- Makefile。
- benchmark 运行日志。
