# Class 1: Buildroot 定位与最小系统构建

## 本章目标

- 理解 Buildroot 的定位。
- 认识工具链、根文件系统、镜像等产物。
- 复现最小系统构建流程。

## 前置条件

- 已完成 Linux 应用开发主线。
- 了解基本交叉编译概念。

## 知识简介

Buildroot 属于系统扩展方向，用于帮助学生理解嵌入式 Linux 系统是如何构建出来的。它不改变全书应用开发主线，也不提前引入内核和驱动负担。本节课只要求学生认识产物、复现最小系统构建，并理解其和应用开发的关系。
