# Lab 3: Device Tree、pinmux 与外设使能入门

## 实验目标

- 阅读一个 Device Tree 外设节点。
- 识别 `compatible`、`status` 和 `pinctrl` 等关键字段。
- 形成应用开发到板级配置的边界说明。

## 实验任务

1. 选择一个 GPIO、UART、I2C 或 SPI 相关 DT 片段。
2. 标注关键属性及其含义。
3. 对照前序实验说明设备文件或接口从何而来。
4. 记录哪些内容属于后续内核/驱动学习范围。
