# Lab 2: 最小系统构建

## 实验目标

- 复现或观察 Buildroot 最小系统构建。
- 记录配置、构建和输出产物。
- 说明构建失败或依赖下载问题。

## 实验任务

1. 选择课程提供的最小配置。
2. 执行或阅读构建流程。
3. 记录构建开始时间、结束时间和关键命令。
4. 查看输出目录。
5. 整理镜像、rootfs、工具链等产物位置。

## 运行验证

| 验证项 | 预期现象 | 是否通过 |
| --- | --- | --- |
| 配置 | 能加载或说明最小配置 |  |
| 构建 | 生成或复现实验镜像 |  |
| 产物 | 能说明各产物用途 |  |
| 问题记录 | 有失败或依赖说明 |  |

## 提交要求

- 构建记录。
- 产物说明。
- 问题记录。
