# Class 1: 面包板、跳线与基础接线

## 本章目标

- 认识面包板、电源轨、连接孔和跳线的基本用法。

## 知识简介

开发板 GPIO 实验不是从代码开始，而是从安全、清晰、可复查的接线开始。面包板提供免焊接的临时连接方式，适合课程中的 LED、按键、传感器和 OLED 等早期实验。接线错误会直接影响后续程序现象：LED 不亮、按键读数不变、GPIO 权限和 line 都正确但实验仍失败，很多时候根因都在接线。

### 面包板连接结构

常见面包板由两类区域组成：

| 区域 | 作用 | 使用注意 |
| --- | --- | --- |
| 电源轨 | 分配 VCC 和 GND | 不同面包板的电源轨可能中间断开，需要确认是否连通 |
| 中央插孔区 | 连接元件和跳线 | 同一短排内部相连，跨中间沟槽不相连 |
| 中间沟槽 | 放置 DIP 元件或分隔两侧电路 | 沟槽两侧默认不导通 |
| 行列标记 | 辅助定位孔位 | 实验记录中可写出元件所在孔位 |

使用面包板前，应先确认电源轨正负方向。红色通常表示 VCC，蓝色或黑色通常表示 GND，但颜色只能作为提示，最终以实际连通性和接线记录为准。

### LED 与限流电阻

LED 是有方向的器件。长脚通常是阳极，短脚通常是阴极；也可以通过外壳平边判断阴极。GPIO 直接驱动 LED 时必须串联限流电阻，避免电流过大损坏 LED 或 GPIO。

基础连接关系可以写成：

```text
GPIO line -> 限流电阻 -> LED -> GND
```

也可能写成：

```text
VCC -> 限流电阻 -> LED -> GPIO line
```

前一种连接通常高电平点亮，后一种连接可能低电平点亮。后续填写 `LED_ACTIVE_LEVEL` 时，必须根据实际接线和现象确认。

### 按键、上拉和下拉

按键本质上是一个临时闭合的开关。没有上拉或下拉时，输入引脚可能处于悬空状态，读数会随机跳变。按键实验需要明确空闲电平和按下电平：

| 接法 | 空闲读数 | 按下读数 |
| --- | --- | --- |
| 上拉电阻 + 按键接地 | 通常为 `1` | 通常为 `0` |
| 下拉电阻 + 按键接 VCC | 通常为 `0` | 通常为 `1` |

如果目标板或扩展板已经提供板载按键，仍然需要查阅板级资料或实测记录，确认按键是否接入 GPIO line、Linux input 子系统，或者被其他驱动占用。

### 共地和电压等级

外接电路必须与开发板共地。没有共地时，GPIO 电平没有共同参考点，LED 和按键现象可能不可预测。GPIO 电压等级也必须匹配目标板要求，不能把 5V 信号直接接到只支持 3.3V 的 GPIO 输入。

## 环境准备

| 项目 | 要求 | 检查方式 |
| --- | --- | --- |
| 面包板 | 电源轨和插孔可用 | 用跳线或万用表检查连通性 |
| 杜邦线 | 公对公或课程指定线材 | 轻拉不松动 |
| LED | 方向可识别 | 长短脚或外壳平边 |
| 限流电阻 | 阻值适合课程实验 | 按器件包或万用表确认 |
| 按键 | 四脚或两脚按键可用 | 用万用表或插孔方向确认 |
| 目标板 | GPIO 电压等级明确 | 查看 `boards/` 公开说明 |

## 操作步骤

### 步骤 1：识别面包板连通区域

用一根跳线或万用表确认：

```text
同一短排是否导通：
中间沟槽两侧是否不导通：
电源轨是否全长导通：
左右电源轨是否需要跳线连接：
```

记录面包板型号或照片编号，方便后续复查。

### 步骤 2：检查 LED 和限流电阻

记录：

```text
LED 颜色：
LED 阳极位置：
LED 阴极位置：
限流电阻阻值：
接线方向：
```

LED 串联限流电阻后再接入 GPIO 回路。不要让 GPIO 直接短接到 VCC 或 GND。

### 步骤 3：检查按键方向

四脚按键通常同侧两脚内部相连，按下时两侧导通。把按键跨面包板中间沟槽放置，可以避免同侧短接导致按键一直导通。

记录：

```text
按键放置孔位：
空闲连通状态：
按下连通状态：
计划采用上拉还是下拉：
```

### 步骤 4：形成接线记录

后续每个 GPIO 实验都应保留接线记录：

```text
板卡：
GPIO 电压等级：
外设：
VCC 接到：
GND 接到：
GPIO line 接到：
限流/上拉/下拉：
风险检查：
```

接线记录与程序参数同等重要。没有接线记录时，后续 LED 不亮或按键不变很难复查。

## 常见问题

| 现象 | 可能原因 | 处理方式 |
| --- | --- | --- |
| LED 不亮 | LED 方向反了、限流电阻未串联、GPIO line 接错 | 断电后复查 LED 方向、孔位和连接关系 |
| LED 常亮 | 接线绕过 GPIO，或有效电平判断错误 | 复查 LED 是接到 GPIO 供电还是 GPIO 拉低 |
| 按键读数随机跳变 | 输入悬空，没有上拉或下拉 | 增加上拉/下拉，或确认板载按键电路 |
| 按键一直导通 | 四脚按键放置方向错误 | 让按键跨中间沟槽放置 |
| 开发板异常重启 | VCC/GND 短接或外设电流过大 | 立即断电，检查电源轨和外设连接 |
