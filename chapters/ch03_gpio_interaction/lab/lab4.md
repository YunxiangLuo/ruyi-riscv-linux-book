# Lab 4: 按键消抖、状态机与可复用交互模块

## 实验目标

- 观察按键抖动。
- 加入消抖逻辑。
- 使用状态机控制 LED 模式。

## 实验环境

| 项目 | 要求 |
| --- | --- |
| 目标环境 | RISC-V Linux 板卡 |
| 硬件连接 | LED 和按键 |
| 软件依赖 | ch03 GPIO 代码工程 |

## 实验任务

### 任务 1：记录原始按键输出

保存按键按下、释放和重复触发记录。

### 任务 2：加入消抖

调整消抖时间并记录效果。

### 任务 3：实现模式切换

按键触发 LED 在不同模式间切换。

## 实验步骤

```bash
cd chapters/ch03_gpio_interaction/code/class4
make build
./build/gpio_state_demo <led_chip> <led_line> <led_active> <led_inactive> <button_chip> <button_line> <button_active>
```
