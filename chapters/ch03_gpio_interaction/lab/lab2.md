# Lab 2: GPIO 输入读取按键

## 实验目标

- 使用 GPIO 输入读取按键状态。
- 记录按键未按下和按下时的电平。
- 完成 Button 原始状态观察。

## 实验任务

1. 检查 `gpioget` 和 GPIO 设备。
2. 填写 `BUTTON_CHIP`、`BUTTON_LINE`、`BUTTON_ACTIVE_LEVEL`。
3. 读取按键未按下和按下状态。
4. 连续观察按键状态变化。
5. 记录是否出现机械抖动现象。

## 运行验证

| 验证项 | 预期现象 | 是否通过 |
| --- | --- | --- |
| Button 参数确认 | 能写出 Button chip、line、有效电平 |  |
| Button 单次读取 | 未按下和按下读数不同 |  |
| Button 连续观察 | 按下和松开时终端输出状态变化 |  |
| 抖动记录 | 若一次按键多次跳变，能记录现象 |  |

## 提交要求

- GPIO 工具检查结果。
- Button GPIO 参数表。
- 未按下和按下读数。
- Button 连续观察日志。
- 抖动或异常现象说明。
