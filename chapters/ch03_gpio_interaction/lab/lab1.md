# Lab 1: 面包板、跳线与基础接线

## 实验目标

- 识别面包板电源轨、中央插孔区和中间沟槽。
- 完成 LED、限流电阻和按键方向检查。
- 建立 GPIO 实验接线记录模板。

## 实验环境

| 项目 | 要求 |
| --- | --- |
| 面包板 | 可插接元件和跳线 |
| 元件 | LED、限流电阻、按键 |
| 工具 | 杜邦线，推荐万用表 |
| 目标板 | K1 / Muse Pi Pro；临时验证可用 LicheePi 4A |

## 实验任务

### 任务 1：检查面包板连通性

确认同一短排、中间沟槽两侧、电源轨和左右电源轨的导通关系。

### 任务 2：检查 LED 回路

识别 LED 阳极、阴极和限流电阻，画出或写出连接关系。

### 任务 3：检查按键方向

确认按键空闲和按下时的导通关系，并记录上拉或下拉方案。

## 实验步骤

```text
1. 给面包板电源轨标记 VCC 和 GND。
2. 用万用表或跳线检查插孔连通关系。
3. 记录 LED 方向和限流电阻阻值。
4. 把按键跨中间沟槽放置，检查按下前后的导通状态。
5. 写出后续 GPIO 实验的接线记录模板。
```

## 运行验证

| 验证项 | 预期现象 | 是否通过 |
| --- | --- | --- |
| 面包板连通性 | 能说明短排、电源轨和中间沟槽的导通关系 |  |
| LED 方向 | 能区分阳极和阴极 |  |
| 限流保护 | LED 回路包含限流电阻 |  |
| 按键方向 | 能说明空闲和按下时的导通关系 |  |
| 接线记录 | 能写出 VCC、GND、GPIO line 和外设连接关系 |  |

## 常见问题

| 现象 | 可能原因 | 处理方式 |
| --- | --- | --- |
| 电源轨不连续 | 面包板电源轨中间断开 | 用跳线桥接并记录 |
| LED 不亮 | LED 方向反或未串限流电阻 | 断电后复查方向和电阻 |
| 按键一直导通 | 按键插入方向错误 | 跨中间沟槽重新放置 |

## 提交要求

- 面包板连通性记录。
- LED 与限流电阻记录。
- 按键方向和上拉/下拉记录。
- 后续 GPIO 实验接线记录模板。
