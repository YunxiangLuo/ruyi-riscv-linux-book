# Class 1: 文件 I/O 与配置文件

## 本章目标

- 使用 Linux 文件 I/O 保存数据。
- 使用配置文件控制程序行为。

## 前置条件

- 已有可产生数据的示例程序。

## 知识简介

文件和配置是程序从示例走向应用的起点。文件 I/O 让程序可以保存数据和复查运行结果；配置文件让采样周期、输出路径等参数不再硬编码。学生应学会用文件组织程序输入和输出。

## 环境准备

| 项目 | 要求 | 检查方式 |
| --- | --- | --- |
| 目标系统 | 可读写文件 | `touch` 测试 |
| 工程模板 | 可运行 | `make build` |

## 操作步骤

1. 写入数据文件。
2. 读取配置文件。
3. 用配置控制程序启动。
