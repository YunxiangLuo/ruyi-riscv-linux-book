# Class 3: 采集与显示闭环

## 本章目标

- 整合采集、处理和显示。
- 完成可工作的本地传感终端。

## 前置条件

- 已完成 PWM、ADC、OLED、DHT22 或同类外设实验。

## 知识简介

本节课把前两节的外设能力合并成局部作品。程序应周期性采集数据，进行必要格式化，并刷新到 OLED 或终端。这个闭环会在后续文件、线程和综合项目中继续复用。

## 环境准备

| 项目 | 要求 | 检查方式 |
| --- | --- | --- |
| 采集模块 | 能输出有效数据 | 终端显示 |
| 显示模块 | 能刷新数据 | OLED 或终端 |

## 操作步骤

1. 定义数据结构。
2. 设置采样周期。
3. 刷新显示内容。
4. 处理异常读数。

| --- | --- | --- |
| 采集 | 周期性得到数据 |  |
| 显示 | 数据稳定刷新 |  |

## 常见问题

- 刷新过快导致闪烁。
- 采样失败未处理。
