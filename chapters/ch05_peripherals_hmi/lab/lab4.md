# Lab 4: DHT22 采集与本地可视化终端

## 实验目标

- 读取 DHT22 或使用替代数据源。
- 将采集值显示到终端或 OLED。
- 记录采集、显示和错误处理结果。

## 实验环境

| 项目 | 要求 |
| --- | --- |
| 目标环境 | RISC-V Linux 开发板 |
| 硬件连接 | DHT22 或替代输入 |
| 软件依赖 | 章节示例工程 |

## 实验任务

### 任务 1：采集数据

读取温湿度或模拟数据。

### 任务 2：显示数据

将数据刷新到终端或 OLED。

### 任务 3：记录异常

断开传感器或模拟错误，记录程序反应。

## 实验步骤

```bash
make build
./build/local_terminal
```
