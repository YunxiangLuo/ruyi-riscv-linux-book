# Class 1: PWM 与 ADC

## 本章目标

- 使用 PWM 控制可见输出。
- 使用 ADC 读取模拟量。

## 前置条件

- 已完成 GPIO 基础实验。
- 已准备 PWM/ADC 可用的开发板或扩展模块。

## 知识简介

PWM 和 ADC 分别代表常见外设中的输出控制和输入采样。PWM 通过占空比改变 LED 亮度或执行器状态；ADC 将模拟输入转换为数字读数。它们不需要展开成复杂协议章，而应服务于“采集—显示”闭环。

| --- | --- | --- |
| PWM 输出 | LED 或可观察负载 | 亮度变化 |
| ADC 输入 | 可变模拟输入 | 数值变化 |
