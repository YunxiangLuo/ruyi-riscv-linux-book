# Lab 3: 串口日志、命令回显与调试入口

## 实验目标

- 输出串口调试日志。
- 实现简单命令回显。
- 记录串口调试过程。

## 实验环境

| 项目 | 要求 |
| --- | --- |
| 目标环境 | RISC-V Linux 开发板 |
| 硬件连接 | 串口线或板载调试串口 |
| 软件依赖 | 串口终端工具 |

## 实验任务

### 任务 1：输出日志

程序周期性输出带标签的日志。

### 任务 2：命令回显

输入 `status`、`help` 等命令并得到响应。

## 实验步骤

```bash
make build
./build/uart_console
```

## 运行验证

| 验证项 | 预期现象 | 是否通过 |
| --- | --- | --- |
| 日志 | 串口能看到运行日志 |  |
| 命令 | 输入命令有响应 |  |

## 常见问题

| 现象 | 可能原因 | 处理方式 |
| --- | --- | --- |
| 串口乱码 | 参数不匹配 | 检查波特率和数据位 |

## 提交要求

- 串口日志记录。
- 命令输入和响应记录。
- 失败排查记录。
