# Class 2: UART Echo 与收发缓冲

## 本章目标

- 编写 UART Echo 程序。
- 理解收发缓冲和阻塞行为。
- 使用串口输出调试日志。

## 前置条件

- 已能打开串口终端。

## 知识简介

串口程序的核心是读写数据和处理缓冲。Echo 程序能帮助学生理解输入、处理、输出的最小闭环。加入日志输出后，串口就成为后续外设和项目调试的重要工具。本节课强调可复用的调试能力。

## 环境准备

| 项目 | 要求 | 检查方式 |
| --- | --- | --- |
| 串口设备 | 可读写 | Echo 测试 |
| 工程模板 | 可编译 | `make build` |

## 操作步骤

1. 编写 UART Echo。
2. 观察阻塞和缓冲行为。
3. 加入日志格式。
4. 实现简单命令回显。
