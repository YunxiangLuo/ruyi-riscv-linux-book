# Class 1: 综合项目模块组装

## 本章目标

- 复用前序章节模块。
- 建立联网信息采集终端项目骨架。

## 前置条件

- 已完成 GPIO、外设、文件、线程基础。

## 知识简介

综合项目不从零开始，而是复用前面章节形成的模块。学生需要把采集、显示、日志和线程调度组合起来，形成一个可维护的项目骨架。本节课重点是模块边界和接口约定。

## 环境准备

| 项目 | 要求 | 检查方式 |
| --- | --- | --- |
| 前序模块 | 可单独运行 | 运行各章节 demo |
| 工程模板 | 可构建 | `make build` |

## 操作步骤

1. 列出可复用模块。
2. 定义项目目录。
3. 定义模块接口。
4. 完成项目骨架编译。
