# Lab 4: HTTP 状态页发布与综合验收

## 实验目标

- 启动联网信息采集终端。
- 访问 HTTP 状态页。
- 完成项目验收记录。

## 实验环境

| 项目 | 要求 |
| --- | --- |
| 目标环境 | RISC-V Linux 开发板 |
| 网络 | 可从主机访问目标板 |
| 软件依赖 | ch08 项目工程 |

## 实验任务

### 任务 1：启动服务

运行项目程序并记录日志。

### 任务 2：访问页面

使用浏览器或 `curl` 查看状态页。

### 任务 3：综合验收

提交截图、日志和说明。

## 实验步骤

```bash
make build
make run
curl http://目标IP:端口/
```
