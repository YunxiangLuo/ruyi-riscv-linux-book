# Class 2: 线程架构、采集、本地显示与日志

## 本章目标

- 设计项目线程架构。
- 集成数据采集、本地显示和日志。

## 前置条件

- 已建立项目骨架。
- 已完成线程和文件日志基础。

## 知识简介

联网信息采集终端需要持续采集数据、刷新本地显示、记录日志，并为网络发布提供最新状态。线程架构可以让这些任务各司其职，日志则提供调试和验收依据。

## 环境准备

| 项目 | 要求 | 检查方式 |
| --- | --- | --- |
| 采集模块 | 可输出数据 | 采集日志 |
| 显示模块 | 可显示状态 | OLED 或终端 |
| 日志模块 | 可写文件 | 日志文件 |

## 操作步骤

1. 划分采集、显示、日志线程。
2. 定义共享状态。
3. 集成数据采集。
4. 刷新本地显示。
5. 写入项目日志。
