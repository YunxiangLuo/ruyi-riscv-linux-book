# Class 1: pthread 任务拆分

## 本章目标

- 创建和回收线程。
- 使用互斥锁保护共享数据。
- 理解基本同步关系。

## 前置条件

- 已有可运行的采集或显示程序。

## 知识简介

多线程让程序同时处理采集、显示、记录等任务，但也会带来共享数据竞争。pthread 是 Linux 应用开发中的基础能力，互斥和同步则是程序稳定运行的关键。本节课先建立线程生命周期和共享数据保护的基础。
