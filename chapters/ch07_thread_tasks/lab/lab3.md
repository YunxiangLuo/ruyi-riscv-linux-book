# Lab 3: 生产者-消费者模型与线程调试

## 实验目标

- 实现一个生产者-消费者队列。
- 使用日志观察线程状态。
- 记录一次同步问题排查过程。

## 实验环境

| 项目 | 要求 |
| --- | --- |
| 主机环境 | 可编译 pthread 程序 |
| 目标环境 | QEMU 或开发板 |
| 软件依赖 | `pthread` |

## 实验任务

### 任务 1：生产数据

采集线程周期性写入队列。

### 任务 2：消费数据

显示或日志线程读取队列。

### 任务 3：调试线程

记录等待、唤醒、队列满或队列空的日志。

## 实验步骤

```bash
make build
make run
```

## 运行验证

| 验证项 | 预期现象 | 是否通过 |
| --- | --- | --- |
| 队列 | 数据持续传递 |  |
| 线程 | 无死锁 |  |
| 日志 | 能解释线程状态 |  |

## 常见问题

| 现象 | 可能原因 | 处理方式 |
| --- | --- | --- |
| 死锁 | 加锁顺序或退出条件错误 | 增加日志定位 |
