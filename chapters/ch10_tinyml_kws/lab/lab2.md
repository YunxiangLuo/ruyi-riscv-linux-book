# Lab 2: 板上 KWS 推理与语音控灯

## 实验目标

- 部署并运行板上 int8 KWS 推理程序。
- 将识别结果映射到 LED 控制。
- 完成“开/关/闪”语音控灯演示。

## 实验任务

1. 编译 KWS 推理程序。
2. 使用预录音频或实时音频运行推理。
3. 输出关键词、置信度和推理耗时。
4. 集成 LED 控制逻辑。
5. 测试“开/关/闪”三类指令。

## 运行验证

| 验证项 | 预期现象 | 是否通过 |
| --- | --- | --- |
| “开” | LED 点亮 |  |
| “关” | LED 熄灭 |  |
| “闪” | LED 闪烁 |  |
| 推理日志 | 有关键词、置信度、耗时记录 |  |

## 提交要求

- 运行日志。
- 语音控灯演示记录。
- 关键词、置信度和耗时记录。
- 异常输入或低置信度处理说明。
