# Lab 2: 语音样本准备与预处理

## 实验目标

- 准备“开/关/闪”语音样本或预录音频。
- 统一样本命名、采样参数和记录格式。
- 标记噪声样本和无效样本。

## 实验任务

1. 建立三类样本目录。
2. 采集或导入“开/关/闪”样本。
3. 记录采样率、时长、样本数量和来源。
4. 进行必要的裁剪、重采样或格式转换。
5. 标记噪声样本、静音样本和无效样本。

## 运行验证

| 验证项 | 预期现象 | 是否通过 |
| --- | --- | --- |
| 样本文件 | 三类关键词样本可读取 |  |
| 命名规则 | 文件名可追溯类别和序号 |  |
| 参数记录 | 采样率、时长和来源完整 |  |
| 质量检查 | 有无效样本处理说明 |  |

## 提交要求

- 样本清单。
- 预处理记录。
- 样本质量问题说明。
