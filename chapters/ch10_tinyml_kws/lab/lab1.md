# Lab 1: KWS 样本准备与 MFCC 导出

## 实验目标

- 准备“开/关/闪”语音样本或预录音频。
- 完成 MFCC 特征提取。
- 导出 int8 模型参数或 `weights.h`。

## 实验任务

1. 建立 KWS 实验目录。
2. 采集或导入“开/关/闪”三类语音样本。
3. 记录采样率、时长、样本数量和命名规则。
4. 运行 MFCC 特征提取。
5. 导出 int8 模型参数或 `weights.h`。

## 运行验证

| --- | --- | --- |
| 样本文件 | 三类关键词样本可读取 |  |
| MFCC | 能输出特征文件或中间结果 |  |
| 模型导出 | 生成 int8 参数或 `weights.h` |  |

## 提交要求

- 样本清单。
- MFCC 或模型导出日志。
- `weights.h` 或等效模型参数文件。
- 环境说明和失败记录。
