# Lab 1: KWS 控灯最小流程

## 实验目标

- 建立 KWS 控灯项目目录。
- 使用模拟关键词或预录样本触发一次 LED 动作。
- 记录真实验证环节和待验证环节。

## 实验任务

1. 创建 `audio/`、`model/`、`src/`、`logs/` 目录。
2. 绘制最小数据流：输入、推理、关键词、LED。
3. 使用模拟输入触发 LED 点亮、熄灭或闪烁中的一种动作。
4. 记录本次实验中哪些内容是真实运行，哪些内容仍待后续验证。

## 运行验证

| 验证项 | 预期现象 | 是否通过 |
| --- | --- | --- |
| 项目目录 | 目录结构清楚 |  |
| 最小触发 | LED 有一次可观察动作 |  |
| 边界记录 | 能区分真实运行和模拟输入 |  |

## 提交要求

- 项目目录说明。
- 最小触发记录。
- 待验证环节清单。
