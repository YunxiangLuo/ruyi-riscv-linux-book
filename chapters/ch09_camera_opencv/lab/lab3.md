# Lab 3: OpenCV 编译链接与基础图像处理

## 实验目标

- 编译 OpenCV 示例。
- 对采集图片做基础处理。
- 保存并对比处理结果。

## 实验环境

| 项目 | 要求 |
| --- | --- |
| 目标环境 | QEMU 或开发板 |
| 软件依赖 | OpenCV 或课程指定替代方案 |
| 输入文件 | 已采集图片 |

## 实验任务

### 任务 1：检查依赖

确认 OpenCV 版本或替代处理工具。

### 任务 2：处理图像

完成灰度、阈值或边缘处理。

### 任务 3：保存结果

保存原图和处理后图片。

## 实验步骤

```bash
make build
./build/opencv_demo input.jpg output.jpg
```
