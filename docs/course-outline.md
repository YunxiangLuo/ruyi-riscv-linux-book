## 主线与扩展关系

- ch01-ch07 的实验必须成为 ch08 联网信息采集终端的基础步骤。
- ch08 只做整合、接口对齐、项目级线程架构、HTTP 状态页和综合验收。
- ch09、ch10、ch11 每章分别形成一个完整扩展项目，不只停留在知识介绍。
- ch10 默认项目是 TinyML KWS 语音关键词识别控灯，Face Detection、YOLO 和大模型 / Agent / Workflow 控制设备只作为备选拓展。
